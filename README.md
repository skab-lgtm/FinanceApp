# Financial Command Center

Personal finance portfolio tracker with transaction history.

## Quick Start (Windows)

```powershell
# Deploy changes
.\deploy.ps1 "your commit message"

# Emergency rollback
.\rollback.ps1

# Sync from Claude
.\sync-from-claude.ps1
```
